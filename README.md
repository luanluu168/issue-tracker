# issue-tracker
### use to track todos, bugs, and features for developers

## To Run
#### 1. Install all dependencies if not do it yet
```
npm i
```
#### 2. Install pm2
```
npm install pm2 -g
```
#### 3.
```
npm start
```
#### Or
```
pm2 start process.config.js
```

## To Stop
```
npm stop
```
#### Or
```
pm2 kill
```
