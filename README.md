## To Run
#### 1. Install all dependencies if not do it yet
```
npm i
```
#### 2. Install pm2 if not do it yet
```
npm install pm2 -g
```
#### 3. Run the app
```
npm test
```
#### Or
```
pm2 start process.config.js
```

## To Stop
```
npm stop
```
#### Or
```
pm2 kill
```

## Microservices in backend: Express Redis
