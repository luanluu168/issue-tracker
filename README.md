# issue-tracker
### use to track todos, bugs, and features

## Requrements

- Node & npm or yarn

## Installation
#### 1. Install dependencies
```
npm i
```
#### 2. Install pm2 globally
```
npm install pm2 -g
```

## Development

#### 3. Start express development server
```
npm test
```
#### Or
```
pm2 start process.config.js
```

#### To Stop
```
npm stop
```
#### Or
```
pm2 kill
```
